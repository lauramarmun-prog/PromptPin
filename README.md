# 🌸 PromptPin

**PromptPin** is a simple, local-first web app to save, organize and reuse your prompts.

✨ No accounts  
✨ No tracking  
✨ Your data stays on your device  

This project is designed to be **easy, lightweight and stress-free**.

---

## 🧠 How does it work?

PromptPin is **local-first**.

That means:
- Your data is saved in **your browser**
- It lives on **your device**
- If you open the app on another device (or clear browser data), it will start empty again

This is intentional, to keep things simple and private 💛

---

## 🚀 Installation (Netlify – no coding required)

### Step 1 — Download the files
1. Go to the GitHub repo:  
   👉 https://github.com/lauramarmun-prog/PromptPin
2. Click **Code → Download ZIP**
3. Remember where you downloaded it
4. **Unzip** the file  
   - You’ll get a folder like `PromptPin-main`

---

### Step 2 — Upload to Netlify
1. Go to https://www.netlify.com/ and create a **free account**
2. Open **Sites**
3. Drag & drop the **unzipped folder** (`PromptPin-main`) into the upload area
4. Wait a few seconds… your website will be live 🎉

---

### Step 3 — Your website
Your site URL will look like:
https://something-random.netlify.app


You can bookmark it and start using PromptPin right away 💫

---

## 🧩 Troubleshooting

- ❌ Blank page or error?  
  → Make sure you uploaded the **unzipped folder**, not the ZIP file.
- ❌ App doesn’t load?  
  → Check that the folder contains an `index.html` file.
